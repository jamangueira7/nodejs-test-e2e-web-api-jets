## 🚀 Como Rodar

 - Clone o projeto.
 - Entre na pasta do projeto e rode o comando "npm ci --silent" para instalar as dependências.
 - Rode "nodemon index.js" para iniciar o servidor.
 - Para testar acesse localhost:3000.
